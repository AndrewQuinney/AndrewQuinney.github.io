Detailed Description

template<std::size_t MAX_SIZE>
struct bezier_geometry::CritsAndValues< MAX_SIZE >

Stores 'critical points' and their associated values.

A critical point refers to a parameter for which a continuous value changes direction, meaning that as the parameter increases, the associated value stops increasing and starts decreasing (or vice versa). These values can refer to coordinates, perpendicular magnitudes of a curve, curve distance from a fixed point, etc.

When referring to any critical values on a Bezier curve, the start and end points of the curve are also considered to be critical points.
