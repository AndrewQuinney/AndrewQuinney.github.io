Detailed Description

template<std::size_t COEFF_COUNT>
class bezier_geometry::PolynomialFunction< COEFF_COUNT >

A single variable polynomial function.

For a given number of constant coefficients (a, b, c, d, e), a polynomial function has the form:

at^4+bt^3+ct^2+dt+e

For a single variable 't'.

Get the degree of this function, excluding sufficiently inconsequential coefficients.

While a function may have higher degree coefficients, they may be so small relative to the values of this function in a given range that it would be inappropriate to consider it to be of that degree. This makes use of the 'constant' logic in this class to determine if a degree is 'effective'.
