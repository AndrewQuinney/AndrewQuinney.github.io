Detailed Description

A single interval on a curve representing a direction of rotation about a fulcrum.

As the Bezier parameter increases a curve may move in a clockwise direction about a given point, counterclockwise, or neither in the case of a straight line pointing at the fulcrum. An instance of this represents a single interval along a curve where the curve moves in a single direction about a point relative to its parameter.
