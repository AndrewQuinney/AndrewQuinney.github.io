#define debug   bezier_geometry::debug_out

Call (or skip) debug output based on compile flags.

All debug output from the module is done by calling 'debug() << ...', which is an invocation of this macro. This effectively becomes a no-op when NDEBUG is defined on compilation, meaning that in addition to no debug output being generated, no parameters to the function are evaluated either.

std::ostream & bezier_geometry::debug_out ( )

Debug output stream.

The module outputs a large volume of debugging data that allows for logging and reproduction of its internal processes. This value represents the output stream that will be used to print this data. By default, this value is STDOUT. This value is not cached and the method is called repeatedly for every debug statement.

Returns
The output stream where debug data is to be written.
